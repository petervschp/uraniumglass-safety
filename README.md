# uraniumglass-safety

Jednoduchá statická stránka (HTML/CSS/JS) pripravená pre GitHub Pages.

## Ako to nasadiť na GitHub Pages

1. Vytvor repozitár: `uraniumglass-safety`
2. Nahraj obsah tohto ZIPu do repozitára (root)
3. V GitHub-e choď na **Settings → Pages**
4. V časti **Build and deployment** nastav:
   - Source: **Deploy from a branch**
   - Branch: **main** (root)
5. Po pár minútach bude stránka dostupná na GitHub Pages.

## Obsah

- `index.html` – hlavná stránka
- `styles.css` – dizajn
- `script.js` – drobný JS (zvýraznenie menu)
- `docs/Uranove-sklo-suhrrn.pdf` – PDF súhrn (pribalené)
- `assets/` – ikonka

## Poznámka

Texty a sekcie sú pripravené tak, aby sa dali ľahko rozšíriť o podstránky:
`database.html`, `register.html`, `quiz.html`, `calculator.html` (pripravíme neskôr).
